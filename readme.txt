# Dynamic Welcome

This plugin personalizes your website's greeting message based on the user and the time of day.

## Features:

- Display personalized greetings for logged-in users.
- Customize greetings for different times of day.
- Display a generic greeting for non-logged-in users.
- Easy to set up and use.

## Installation:

1. Upload the plugin directory to your WordPress plugins folder.
2. Activate the plugin in your WordPress admin panel.

## Usage:

Use the shortcode `[dynamic_welcome]` in any post or page to display the personalized greeting message.

## Example:

[dynamic_welcome]

## Notes:

- Requires WordPress version 4.7 or higher.
- Clear any caching plugins after installation or update.
